# Components

**Design des composants, morceaux d'UI**  
C'est ici que l'on mets les styles principaux pour nos composants.

Examples :

```css
footer 
    h2 
        display: none;
    
    ul 
        list-style: none;

        font-size: 0;

            display: inline-block;
            width: 25%;
```
